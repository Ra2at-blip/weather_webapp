# Weather App 🌤️

A simple weather application that allows users to search for a city and view its current weather information.

## Features

* Search weather by city name
* Display temperature
* Display humidity
* Display wind speed
* Dynamic weather icons
* Error message for invalid city names

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

## How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Enter a city name and click the search button.

## How it works

1. User enters a city name.
2. JavaScript sends a request to the OpenWeatherMap API.
3. Weather information is fetched and displayed.
4. Appropriate weather icons are shown based on current weather conditions.
5. If the city name is invalid, an error message is displayed.
